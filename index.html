Git Tutorial on this Video

Notes

Start the git process by typing "git init" in the command line of your project.

"git status" Will show what current files have been changed and tracked

"git add [filename] OR "git add ."(to select all files)" will add the files to be ready for committing.

"git commit -m "[your commit message]"" Saves the current changes along with a message

"git log" Shows a list of commits with date/time and message. Press "Q" to end the log.

"git checkout [commit hash key]" Reverts all your changes to the commit chosen by its hash key.

Branches are separate versions of the project where you can work on and commit without interferring with the main/master branch.

"git branch" Shows what branches you have in the repository.

"git branch [branch name]" Adds a new branch to the repository.

"git checkout [branch name]" Moves to the code instance in the selected branch.

"git merge [branch name]" Merges the selected branch into the master/main branch.
